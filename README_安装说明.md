# 申论 AI 提分训练台 PWA

这是一个手机优先的单机/PWA 版本。

## 最简单使用

直接双击 `index.html` 可以使用大部分功能，数据保存在当前浏览器本地。

## 想安装成“像 App 一样”的桌面/手机应用

PWA 安装需要通过 `http://localhost` 或 HTTPS 访问，不能只用 `file://` 双击完成完整安装。

### 电脑本地预览

如果电脑安装了 Python，在本文件夹打开命令行，运行：

```bash
python -m http.server 8899
```

然后浏览器打开：

```text
http://localhost:8899
```

Chrome / Edge 地址栏右侧会出现“安装应用”按钮，或在菜单中选择“安装此网站为应用”。

### 手机上使用

把本文件夹部署到 HTTPS 静态网站后，用手机浏览器打开网址，再选择“添加到主屏幕”。

数据会保存在手机浏览器本地数据库中。清理浏览器数据可能会删除记录，所以建议定期在“导出”页导出 JSON 备份。

## 数据说明

- 档案、训练记录、对话等数据保存在本设备本地。
- 电脑和手机之间不会自动同步。
- 需要换设备时，请先导出 JSON 备份，再在另一台设备导入。

## API 安全

DeepSeek API Key 保存在本地浏览器中，仅适合个人自用。不要把填了自己 API Key 的版本发给别人。
